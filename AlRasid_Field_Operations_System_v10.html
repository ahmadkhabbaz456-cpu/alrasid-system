Connect Firebase Database
One-time setup · Takes 5 minutes
Follow these steps:
  1. Go to console.firebase.google.com
  2. Click "Add project" → name it "AlRasid" → Continue
  3. Click "Build" → "Realtime Database" → Create database → Start in test mode
  4. Click "Project Settings" (gear icon) → "Your apps" → Web icon (</>) → Register app
  5. Copy the values from the firebaseConfig object and paste below
  6. In Realtime Database, go to Rules and set both read/write to true
